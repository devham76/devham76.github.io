## 문제유형
- 다익스트라 알고리즘
## 해결
- 추의 순서대로 정렬한다
- 모든 추를 더한 무게 <  다음 추의 무게 이면 모든추를 더한 무게+1의 공백이 생긴다

## 다른 사람 풀이
```java
import java.util.*;

class Solution {
    public int solution(int[] weight) {
        Arrays.sort(weight);

        int ans = 1;
        for (int e : weight) {
            if (ans < e)
                break;
            ans += e;
        }

        return ans;
    }
}
```
